初识 React Native

安装 由于开发需要，稍稍玩了玩 React Native 开发一个 App 的简要流程。 官网 安装使用时，可以使用 Expo ，可以省去在 Xcode 或者 Android Studio 配置的过程。 在 macOS 上，运行以下命令安装 Expo npm install -g expo-cli 注意这里在 npm 前不要加 sudo，请提前确保权限通顺 sudo chown -R $(whoami) ~/.npm sudo chown -R $(whoami) /usr/local/lib/node_modules 安装好后，我试图运行 Expo，但是得到了 Expo 不支持当前的 Node 版本的错误，于是运行 nvm 来安装最新版本 Node（以下是未安装 nvm 的版本）： sudo npm install -g n sudo n latest 开始 安装好后，生成第一个项目： expo init rn_proj 进入目录并运行 cd rn_proj expo start 然后在待调试的设备上下载 Expo Go 后扫码即可开始调试...
